개인정보처리방침

시행일: 2026년 3월 25일

1. 수집하는 개인정보

나갈때 앱은 다음 정보를 처리합니다:

  • 위치 정보 — 현재 위치의 날씨 정보를 조회하기 위해 사용됩니다
  • 알림 토큰 — 매일 아침 날씨 알림을 발송하기 위해 사용됩니다
  • 사용자 설정 — 출퇴근 시간, 알림 시간, 선택한 도시 등

2. 개인정보의 저장

모든 데이터는 사용자의 기기 내(AsyncStorage)에 저장됩니다.

  • 별도의 외부 서버에 개인정보를 저장하지 않습니다
  • 위치 정보는 날씨 API 호출 시에만 일시적으로 사용되며, 서버에 저장되지 않습니다
  • 개발자는 사용자의 위치 정보에 접근할 수 없습니다

3. 제3자 제공

날씨 및 대기질 정보를 제공하기 위해 다음 외부 서비스를 이용합니다:

  • Open-Meteo (기상청 KMA 모델) — 날씨 데이터 조회 (위도/경도 전송)
  • 에어코리아 — 실시간 대기질 데이터 조회 (측정소명 전송)

상기 서비스 외에 개인정보를 제3자에게 제공하지 않습니다.

4. 데이터 삭제

  • 앱을 삭제하면 기기에 저장된 모든 데이터가 삭제됩니다
  • 설정 화면에서 위치 모드를 변경하여 위치 데이터 사용을 중단할 수 있습니다

5. 아동 보호

본 앱은 만 14세 미만 아동의 개인정보를 의도적으로 수집하지 않습니다.

6. 변경 사항

본 방침이 변경될 경우 앱 업데이트를 통해 안내드립니다.

7. 문의

Privacy Policy

Effective: March 25, 2026

1. Information We Collect

Nagalttae processes the following information:

  • Location data — Used to fetch weather information for your current location
  • Push notification token — Used to send daily morning weather notifications
  • User preferences — Commute times, notification time, selected city, etc.

2. Data Storage

All data is stored locally on your device (AsyncStorage).

  • We do not store personal information on external servers
  • Location data is used only temporarily for weather API calls and is not stored on any server
  • The developer cannot access your location information

3. Third-Party Services

We use the following external services to provide weather and air quality information:

  • Open-Meteo (KMA model) — Weather data retrieval (latitude/longitude transmitted)
  • AirKorea — Real-time air quality data retrieval (station name transmitted)

We do not share personal information with any third parties beyond the services listed above.

4. Data Deletion

  • Deleting the app removes all data stored on the device
  • You can stop location data usage by changing the location mode in Settings

5. Children's Privacy

This app does not knowingly collect personal information from children under 14.

6. Changes

Any changes to this policy will be communicated through app updates.

7. Contact

Privacy inquiries: cozynye@gmail.com
